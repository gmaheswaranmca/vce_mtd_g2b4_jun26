## 394. Decode String
**Difficulty:** Medium

### Description
```text
Given an encoded string, return its decoded string.
The encoding rule is:`k[encoded_string]`
where the `encoded_string` inside the square brackets is repeated exactly `k` times.
You may assume:
    * `k` is a positive integer.
    * The input string is always valid.
    * No white spaces exist.
    * Digits are only used as repeat counts.
```
---

### Examples

```text
Input: s="3[a]2[bc]"
Output: "aaabcbc"
Explanation:
3[a] -> "aaa"
2[bc] -> "bcbc"
Result -> "aaabcbc"
```

```text
Input: s="3[a2[c]]"
Output: "accaccacc"
Explanation:
a2[c] -> "acc"
3[acc] -> "accaccacc"
```

```text
Input: s="2[abc]3[cd]ef"
Output: "abcabccdcdcdef"
Explanation:
2[abc] -> "abcabc"
3[cd] -> "cdcdcd"
Append ef.
```

---

### Constraints

```text
1<=s.length<=30
s consists of lowercase English letters,digits,and '[]'.
1<=k<=300
Input is always valid.
Decoded string length<=10^5
```

---

## Idea (One Line)
Use two stacks to save previous string and repeat count whenever `[` is encountered, then rebuild when `]` appears.


## Pseudocode
```text
function decodeString(s)
    numStk=emptyStack()
    striStk=emptyStack()

    stri=""
    num=0
    for each ch in s
        if isDigit(ch)
            num=num*10+(ch-'0')
        else if ch=='['
            push num into numStk
            push stri into striStk
            num=0
            stri=""
        else if ch==']'
            prevNum=pop numStk
            prevStri=pop striStk
            stri=prevStri+repeat(stri,prevNum)
        else
            stri=stri+ch
    return stri
```

---

## Complexity
```text
Time Complexity: O(n+m)
n=length of input string
m=length of decoded output
Space Complexity: O(n)
```

## Code Explanation
```text
1. Traverse each character from left to right.
2. If digit appears, build repeat count using currentNumber=currentNumber*10+digit.
3. If '[' appears, save currentNumber into countStack.
4. Save currentString into stringStack.
5. Reset currentNumber=0 and currentString="" for inner substring.
6. If alphabet appears, append to currentString.
7. If ']' appears, pop repeatCount from countStack.
8. Pop previousString from stringStack.
9. Repeat currentString repeatCount times.
10. Concatenate previousString with repeated string.
11. Continue until traversal finishes.
12. Return currentString.
```

# Other Patterns
## Pattern 1: Two Stacks (Standard)
### Idea
Store repeat count and previous string separately using two stacks.

### Pseudocode
```text
function decodeString(s)
    countStack=emptyStack()
    stringStack=emptyStack()
    currentString=""
    currentNumber=0
    for each ch in s
        if isDigit(ch)
            currentNumber=currentNumber*10+(ch-'0')
        else if ch=='['
            push currentNumber into countStack
            push currentString into stringStack
            currentNumber=0
            currentString=""
        else if ch==']'
            count=pop countStack
            previous=pop stringStack
            currentString=previous+repeat(currentString,count)
        else
            currentString=currentString+ch
    return currentString
```
---
## Pattern 2: Single Stack
### Idea
Push characters one by one, when `]` appears, pop until `[` and rebuild substring.
### Pseudocode
```text
function decodeString(s)
    stack=emptyStack()
    for each ch in s
        if ch!=']'
            push ch into stack
        else
            temp=""
            while top(stack)!='['
                temp=pop(stack)+temp
            pop stack
            number=""
            while stack not empty and isDigit(top(stack))
                number=pop(stack)+number
            count=toInteger(number)
            expanded=repeat(temp,count)
            for each c in expanded
                push c into stack

    answer=""
    while stack not empty
        answer=pop(stack)+answer
    return answer
```

## Pattern 3: Recursive DFS
### Idea
When `[` appears, recursively decode inner substring and repeat it.

### Pseudocode
```text
function decodeString(s)
    index=0
    return dfs(s,index)

function dfs(s,index)
    result=""
    number=0
    while index<length(s) and s[index]!=']'
        if isDigit(s[index])
            number=number*10+(s[index]-'0')
        else if s[index]=='['
            index=index+1
            inner=dfs(s,index)
            result=result+repeat(inner,number)
            number=0
        else
            result=result+s[index]
        index=index+1
    return result
```

# Comparison
| Pattern        | Data Structure |   Time | Space | Remarks                         |
| -------------- | -------------- | -----: | ----: | ------------------------------- |
| Two Stacks     | 2 Stacks       | O(n+m) |  O(n) | Most popular interview solution |
| Single Stack   | 1 Stack        | O(n+m) |  O(n) | Compact stack based approach    |
| Recursive DFS  | Recursion      | O(n+m) |  O(n) | Elegant recursive parsing       |
| Queue + Parser | Queue          | O(n+m) |  O(n) | Used in parser implementations  |

**Recommended for Interviews:** **Two Stacks**, because it is easy to understand, naturally handles nested brackets, and is the most commonly expected solution for this problem.
