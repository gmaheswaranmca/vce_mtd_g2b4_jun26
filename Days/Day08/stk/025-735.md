## 025. Asteroid Collision
| Item       | Value                    |
| ---------- | ------------------------ |
| LeetCode   | 735                      |
| Difficulty | Medium                   |
| Pattern    | Stack                    |
| Tags       | Array, Stack, Simulation |

### Description
```text
We are given an array `asteroids` of integers representing asteroids in a row.
* Absolute value represents the size.
* Positive value means moving right.
* Negative value means moving left.

Collision occurs only when:
left asteroid > 0
right asteroid < 0

After collision:
* Smaller asteroid explodes.
* If equal sizes, both explode.
* Asteroids moving in the same direction never collide.

Return the state of asteroids after all collisions.
```
---

### Example 1

```text
Input: asteroids=[5,10,-5]
Output: [5,10]
Explanation:
10 and -5 collide.
|10| > |-5|
-5 explodes.
Remaining=[5,10]
```

---

### Example 2

```text
Input: asteroids=[8,-8]
Output:[]

Explanation:
|8|=|-8|
Both explode.
```

---

### Example 3

```text
Input: asteroids=[10,2,-5]
Output: [10]
Explanation:
2 and -5 collide -> 2 explodes
10 and -5 collide -> -5 explodes
Remaining=[10]
```

---

### Constraints

```text
2<=asteroids.length<=10^4
-1000<=asteroids[i]<=1000
asteroids[i]!=0
```

---

## Idea (One Line)
Use a stack to keep surviving asteroids and repeatedly resolve collisions with the incoming asteroid.

## Pseudo Code
```text
function asteroidCollision(asteroids)
    stack=emptyStack()
    for each asteroid in asteroids
        alive=true
        while alive and asteroid<0 and stack not empty and top(stack)>0
            if top(stack)<abs(asteroid)
                pop stack
            else if top(stack)==abs(asteroid)
                pop stack
                alive=false
            else
                alive=false
        if alive
            push asteroid into stack
    return stack
```

---

## Complexity
```text
Time Complexity: O(n)
Space Complexity: O(n)
```

Reason:
```text
Each asteroid is pushed once and popped at most once.
```

# Code Explanation
```text
1. Create an empty stack.
2. Traverse each asteroid.
3. Positive asteroid can be pushed directly.
4. Negative asteroid may collide with positive asteroids in stack.
5. While collision condition exists:
   - If top is smaller, pop it.
   - If equal, pop top and destroy current asteroid.
   - If top is larger, destroy current asteroid.
6. If current asteroid survives, push it.
7. Return the stack.
```

---

# Why Collision Condition Is
```text
asteroid<0 and top(stack)>0
Because:
Stack top : moving right (+)
Incoming : moving left (-)
      →
      10
          -5
          ←
They move towards each other.
```

### These Never Collide
```text
[-5,-2]
←   ←
Same direction

[5,10]
→   →
Same direction

[-5,10]
←   →
Moving away
```

# Other Patterns
## Pattern 1: Simulation by Repeated Scanning
### Idea
Keep scanning the array and remove colliding pairs until no collision remains.

### Pseudo Code
```text
function asteroidCollision(asteroids)
    changed=true
    while changed
        changed=false
        result=emptyList()

        while i<length(asteroids)
            if collisionExists

```

---

### Complexity
```text
Time Complexity: O(n²)

```

## Pattern 2: Linked List Simulation
### Idea
Store asteroids in doubly linked list and remove colliding neighbors.

### Pseudo Code
```text
function asteroidCollision(asteroids)
    create linked list
    while collision exists
        remove smaller asteroid
    return remaining nodes
```

---

### Complexity
```text
Time Complexity: O(n²)
Space Complexity: O(n)
```

---

## Pattern 3: Recursion
### Idea
Recursively process collisions with previous asteroid.

### Pseudo Code
```text
function collide(stack,asteroid)
    if stack empty
        push asteroid
        return
    if collision
        resolve collision recursively
    else
        push asteroid
```

---

### Complexity
```text
Time Complexity: O(n)
Space Complexity: O(n)
```

# Why Stack Is Natural Here
```text
Suppose: [5,10,-5,-8]
When -5 comes,
it only collides with 10 (latest survivor).
When -8 comes,
it again collides with latest survivor.
This is exactly:
Last Survived -> First Collision
LIFO -> Stack
```

---

## Recommended
```text
Use Stack.
Reason:
1. Collision always occurs with latest surviving asteroid.
2. Each asteroid is pushed once and popped at most once.
3. O(n) time complexity.
4. Standard and most accepted solution for LeetCode 735.
```

## Python
```python
class Solution:
    def asteroidCollision(self, asteroids):
        stack = []
        for asteroid in asteroids:
            alive = True
            while alive and asteroid < 0 and stack and stack[-1] > 0:
                if stack[-1] < abs(asteroid):
                    stack.pop()
                elif stack[-1] == abs(asteroid):
                    stack.pop()
                    alive = False
                else:
                    alive = False
            if alive:
                stack.append(asteroid)
        return stack
```

class Solution {

                    stack.remove(stack.size()-1);

## Go
```go
func asteroidCollision(asteroids []int) []int {
	stack := []int{}
	for _, asteroid := range asteroids {
		alive := true
		for alive && asteroid < 0 && len(stack) > 0 && stack[len(stack)-1] > 0 {
			top := stack[len(stack)-1]
			if top < -asteroid {
				stack = stack[:len(stack)-1]
			} else if top == -asteroid {
				stack = stack[:len(stack)-1]
				alive = false
			} else {
				alive = false
			}
		}

		if alive {
			stack = append(stack, asteroid)
		}
	}

	return stack
}
```

int* asteroidCollision(int* asteroids,int asteroidsSize,int* returnSize){
    int* stack=(int*)malloc(sizeof(int)*asteroidsSize);
    int top=-1;
    for(int i=0;i<asteroidsSize;i++){
        int asteroid=asteroids[i];
        int alive=1;

        while(alive && asteroid<0 && top>=0 && stack[top]>0){
            if(stack[top] < -asteroid){
                top--;
            }
            else if(stack[top] == -asteroid){
                top--;
                alive=0;
            }
            else{
                alive=0;
            }
        }

        if(alive){
            stack[++top]=asteroid;
        }
    }

    *returnSize=top+1;
    return stack;
}
