# Product of Array Except Self

## Difficulty
Medium

## Description
Given an array nums of n integers, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i]. Do this in O(n) time and without using division.

## Examples

### Example 1

```text
Input: [1,2,3,4]
Output: [24,12,8,6]
Explanation: Each output[i] is product of all elements except nums[i].
```

### Example 2

```text
Input: [0,4,0]
Output: [0,0,0]
Explanation: Two zeros make every position zero.
```

## Constraints

```text
2 <= n <= 10^5
-10^9 <= nums[i] <= 10^9
Do not use division; achieve O(n) time and O(1) extra space (excluding output).
```

---

## Idea
Compute prefix products into output then multiply by a running suffix product in a reverse pass.

---

## Pseudo Code

```text
function productExceptSelf(nums)
    n := length(nums)
    result := array of size n filled with 1
    prefix := 1
    for i from 0 to n-1
        result[i] := prefix
        prefix := prefix * nums[i]
    suffix := 1
    for i from n-1 down to 0
        result[i] := result[i] * suffix
        suffix := suffix * nums[i]
    return result
```

---

## Complexity

| Metric           | Value  |
| ---------------- | ------ |
| Time Complexity  | O(n)   |
| Space Complexity | O(1) (output not counted) |

---

## Code Explanation

### Dry Run

| Step | State | Operation | Result |
| ---- | ----- | --------- | ------ |
| 1    | prefix=1,suffix=1 | init | [1,1,1,1] |
| 2    | after left pass | build prefix-products: i=0..3 => result=[1,1,2,6], prefix=24 | [1,1,2,6] |
| 3    | start right pass | suffix=1 | [1,1,2,6] |
| 4    | i=3 | result[3]=6*1; suffix*=4 => suffix=4 | [1,1,2,6] |
| 5    | i=2 | result[2]=2*4; suffix*=3 => suffix=12 | [1,1,8,6] |
| 6    | i=1 | result[1]=1*12; suffix*=2 => suffix=24 | [1,12,8,6] |
| 7    | i=0 | result[0]=1*24; suffix*=1 => suffix=24 | [24,12,8,6] |

1. Initialize result with 1s and prefix=1.
2. Left-to-right pass: set result[i]=product of elements left of i and update prefix.
3. Right-to-left pass: multiply result[i] by suffix (product of elements right of i) and update suffix.
4. Return result.

## Other Patterns

### Pattern 1

**Idea**
Use two auxiliary arrays left[] and right[] holding prefix and suffix products.

**Pseudo Code**

```text
function productExceptSelf(nums)
    n := length(nums)
    left := array of size n
    right := array of size n
    result := array of size n
    left[0] := 1
    for i from 1 to n-1
        left[i] := left[i-1] * nums[i-1]
    right[n-1] := 1
    for i from n-2 down to 0
        right[i] := right[i+1] * nums[i+1]
    for i from 0 to n-1
        result[i] := left[i] * right[i]
    return result
```

### Pattern 2

**Idea**
Use total product and division when guaranteed no zeros.

**Pseudo Code**

```text
function productExceptSelfDivision(nums)
    total := 1
    for v in nums
        total := total * v
    result := array of size length(nums)
    for i from 0 to length(nums)-1
        result[i] := total / nums[i]
    return result
```

## Comparison

| Approach            | Idea                                | Time  | Space             | Recommended |
| ------------------- | ----------------------------------- | ----- | ----------------- | ----------- |
| Prefix-Suffix (two-pass) | Compute left in result then multiply by running right | O(n)  | O(1) (excl. output) | Yes        |
| Left & Right arrays  | Store both prefix and suffix arrays | O(n)  | O(n)              | No         |
| Division (total product) | Use division with total product when no zeros | O(n)  | O(1)              | No (unsafe if zeros) |
