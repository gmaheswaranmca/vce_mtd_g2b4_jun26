## 024. Removing Stars From a String
| Item       | Value                     |
| ---------- | ------------------------- |
| LeetCode   | 2390                      |
| Difficulty | Medium                    |
| Pattern    | Stack                     |
| Tags       | String, Stack, Simulation |

### Description
```text
You are given a string `s`, which contains lowercase English letters and stars `'*'`.

In one operation:
    * Choose a star `'*'`.
    * Remove the closest non-star character to its left.
    * Remove the star itself.

Return the string after all stars have been removed.
It is guaranteed that:
    * The operation is always possible.
    * The resulting string is unique.
```

---

### Example 1
```text
Input:s="leet**cod*e"
Output:"lecoe"
Explanation:
leet**cod*e
Remove t and *
lee*cod*e
Remove e and *
lcod*e
Remove d and *
lecoe
```

---

### Example 2

```text
Input:s="erase*****"
Output:""
Explanation:
erase*****
All characters are removed.
```

---

### Constraints

```text
1<=s.length<=10^5
s consists of lowercase English letters and '*'
The operation is always possible.
```

---

## Idea (One Line)
Use a stack; push letters and pop the top element whenever a `'*'` is encountered.

## Pseudo Code

```text
function removeStars(s)
    stack=emptyStack()
    for each ch in s
        if ch!='*'
            push ch into stack
        else
            pop stack
    answer=join(stack)
    return answer
```

---

## Complexity

```text
Time Complexity: O(n)
Space Complexity: O(n)
```
where `n=s.length`.

# Code Explanation
```text
1. Create an empty stack.
2. Traverse each character in the string.
3. If the character is not '*', push it into stack.
4. Otherwise pop one element from stack.
5. Continue until the end of the string.
6. Join all characters in stack.
7. Return the final string.
```

# Other Patterns
## Pattern 1: String Builder as Stack
### Idea
Use a dynamic string and remove the last character when `'*'` appears.

### Pseudo Code
```text
function removeStars(s)
    result=emptyStringBuilder()
    for each ch in s
        if ch!='*'
            append ch to result
        else
            removeLastCharacter(result)
    return result.toString()
```

**Complexity**
```text
Time Complexity: O(n)
Space Complexity: O(n)
```

---

## Pattern 2: Two Pointers In-place
### Idea
Use write pointer as stack top and overwrite characters in-place.

### Pseudo Code
```text
function removeStars(s)
    arr=toCharArray(s)
    write=0
    for read=0 to length(arr)-1
        if arr[read]!='*'
            arr[write]=arr[read]
            write=write+1
        else
            write=write-1
    return substring(arr,0,write)
```

**Complexity**
```text
Time Complexity: O(n)
Space Complexity: O(1) extra
```

---

## Pattern 3: Explicit Stack using Array
### Idea
Store characters in an array and treat the end as stack top.

### Pseudo Code
```text
function removeStars(s)
    stack=emptyArray()
    top=-1
    for each ch in s
        if ch!='*'
            top=top+1
            stack[top]=ch
        else
            top=top-1
    return charactersFrom(stack,0,top)
```

**Complexity**
```text
Time Complexity: O(n)
Space Complexity: O(n)
```

## Which Pattern Should You Use?
```text
1. Stack -> Easiest to understand and explain.
2. String Builder -> Most common in Java/C# interviews.
3. Two Pointers -> Best optimized solution with O(1) extra space.
4. Explicit Array Stack -> Useful when implementing stack manually.
```

---

## Recommended
```text
Use Stack Simulation.
Reason:
1. Natural interpretation of "remove previous character".
2. Simple implementation.
3. O(n) time.
4. Accepted and widely used solution for LeetCode 2390.
```

## Python
```python
class Solution:
    def removeStars(self, s: str) -> str:
        stack = []

        for ch in s:
            if ch != '*':
                stack.append(ch)
            else:
                stack.pop()
        return "".join(stack)
```

## Java

class Solution {
    public String removeStars(String s) {
        StringBuilder stack = new StringBuilder();
        for(char ch:s.toCharArray()) {
            if(ch!='*') {
                stack.append(ch);
            }
            else {
                stack.deleteCharAt(stack.length()-1);
            }
        }

        return stack.toString();
    }
}

## C
```c
char* removeStars(char* s) {
    int n = strlen(s);
    char* stack = (char*)malloc((n + 1) * sizeof(char));
    int top = -1;
    for(int i=0;i<n;i++) {
        if(s[i] != '*') {
            stack[++top] = s[i];
        }
        else {
            top--;
        }
    }
    stack[top + 1] = '\0';
    return stack;
}
```
