# 015. LC #1456 - Maximum Number of Vowels in a Substring of Given Length

| Field               | Value                                                                                                                      |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **LC#**             | 1456                                                                                                                       |
| **Title**           | Maximum Number of Vowels in a Substring of Given Length                                                                    |
| **Difficulty**      | Medium                                                                                                                     |
| **Description**     | Given a string `s` and an integer `k`, return the maximum number of vowel letters in any substring of `s` with length `k`. |
| **Vowels**          | `a,e,i,o,u`                                                                                                                |
| **Idea (One Line)** | Maintain a fixed-size sliding window and count vowels entering and leaving the window.                                     |

---

## Example 1

```text
Input: s="abciiidef", k=3
Output: 3
Explanation:
Window "abc" -> vowels=1
Window "bci" -> vowels=1
Window "cii" -> vowels=2
Window "iii" -> vowels=3
Window "iid" -> vowels=2
Window "ide" -> vowels=2
Window "def" -> vowels=1
Maximum vowels=3
```

## Constraints

```text
1 <= s.length <= 10^5
s consists of lowercase English letters
1 <= k <= s.length
```

---

# Main Pattern

## Fixed Size Sliding Window

### Idea

```text
Count vowels in first window of size k.
Slide window by removing left character and adding right character.
Track maximum vowel count.
```

---

# Helper Function

```text
isVowel(ch)=ch=='a' or ch=='e' or ch=='i' or ch=='o' or ch=='u'
```

---

# Pseudo Code

```text
function maxVowels(s,k)
    count = 0
    for i = 0 to k-1
        if isVowel(s[i])
            count = count + 1
    maxCount = count
    for i = k to length(s)-1
        if isVowel(s[i-k])
            count = count - 1
        if isVowel(s[i])
            count = count + 1
        maxCount = max(maxCount,count)
        if maxCount == k
            return maxCount 
    return maxCount
```

# Code Explanation

```text
1. Count vowels in the first window of size k.
2. Store it in maxCount.
3. Slide the window one position at a time.
4. If outgoing character is vowel,count--.
5. If incoming character is vowel,count++.
6. Update maxCount=max(maxCount,count).
7. Continue until end of string.
8. Return maxCount.
```

# Complexity

| Operation | Complexity |
| --------- | ---------: |
| Time      |       O(n) |
| Space     |       O(1) |

Reason:

```text
Each character enters the window once and leaves the window once.
```

# Other Patterns

## Pattern 1 : Prefix Sum

### Idea

```text
Store cumulative vowel counts and compute vowels in any window in O(1).
```

### Pseudo Code

```text
function maxVowels(s,k)
    n = length(s)
    prefix[0] = 0
    for i = 0 to n-1
        prefix[i+1] = prefix[i]
        if isVowel(s[i])
            prefix[i+1] = prefix[i+1] + 1
    maxCount = 0
    for i = 0 to n-k
        count = prefix[i+k]-prefix[i]
        maxCount = max(maxCount,count)
    return maxCount
```

---

## Pattern 2 : Queue Based Sliding Window

### Idea

```text
Maintain k characters in a queue and track vowel count.
```

### Pseudo Code

```text
function maxVowels(s,k)
    create queue
    count = 0
    maxCount = 0
    for each ch in s
        push ch
        if isVowel(ch)
            count++
        if size(queue) > k
            out = front(queue)
            pop front
            if isVowel(out)
                count--
        if size(queue) == k
            maxCount = max(maxCount,count)
    return maxCount
```

---

## Pattern 3 : Brute Force

### Idea

```text
Check every substring of size k and count vowels separately.
```

### Pseudo Code

```text
function maxVowels(s,k)
    maxCount = 0
    for i = 0 to length(s)-k
        count = 0
        for j = i to i+k-1
            if isVowel(s[j])
                count++
        maxCount = max(maxCount,count)
    return maxCount
```

# Comparison of Approaches

| Pattern              |   Time | Space | Recommended       |
| -------------------- | -----: | ----: | ----------------- |
| Fixed Sliding Window |   O(n) |  O(1) | ✅ Best            |
| Prefix Sum           |   O(n) |  O(n) | Good              |
| Queue Sliding Window |   O(n) |  O(k) | Good              |
| Brute Force          | O(n×k) |  O(1) | ❌ Not Recommended |

# Important Sliding Window Formula

```text
If outgoing char is vowel,count--
If incoming char is vowel,count++
maxCount=max(maxCount,count)
```

---

# Interview Takeaway

```text
Pattern: Fixed Size Sliding Window

1. Count vowels in first k characters.
2. Slide window by one position.
3. Remove outgoing vowel if present.
4. Add incoming vowel if present.
5. Update answer.
6. Return maxCount.

Time=O(n)
Space=O(1)
```

This is a classic **Fixed Size Sliding Window + Character Counting** problem and is one of the most important sliding window patterns for strings.
