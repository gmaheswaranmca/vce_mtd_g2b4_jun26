# 283. Move Zeroes

Move Zeroes

## Problem Statement

Given an integer array `nums`, move all `0`s to the **end** of the array while maintaining the **relative order of the non-zero elements**.

You must do this:

```text
In-place
```

which means:

* Modify the original array.
* Do not create another array for the answer.

---

## Description

You need to:

1. Keep all non-zero numbers in their original order.
2. Move every zero to the end.
3. Do it in-place.

---

# Approach 1 (Recommended)

## Slow and Fast Pointers

---

### Idea

```
`slow` represent: position to place next non-zero
`fast` represent: current scanning position
When nums[fast] != 0
	-> swap: nums[slow] and nums[fast]
	   then: slow++
```

Let:

```text
slow
```

represent:

```text
position to place next non-zero
```

and

```text
fast
```

represent:

```text
current scanning position
```

---

When:

```text
nums[fast] != 0
```

swap:

```text
nums[slow]

and

nums[fast]
```

then:

```text
slow++
```

# Pseudo Code

```text
function moveZeroes(nums):
    slow = 0
    for fast from 0 to n-1:
        if nums[fast] != 0:
            swap(nums[slow], nums[fast])
            slow++
    return
```

## Pseudo Code

```text
index=0

for x in nums:

    if x!=0:

        nums[index]=x

        index++

    nums[index]=0

    index++
```

* Order preserved

## 1. Brute Force - Repeated Shifting

**Idea:** Whenever zero is found, shift all remaining elements left.

```text
for i = 0 to n-1
    if nums[i] == 0
        for j = i+1 to n-1
            nums[j-1] = nums[j]
        nums[n-1] = 0
        i = i - 1
```

### Complexity

```text
Time  : O(n²)
Space : O(1)
```

---

## 2. Extra Array

**Idea:** Copy non-zero elements first, then fill remaining with zero.

```text
create temp array

for each num in nums
    if num != 0
        append num to temp

while size(temp) < n
    append 0 to temp

copy temp to nums
```

### Complexity

```text
Time  : O(n)
Space : O(n)
```

---

## 3. Two Pass Two Pointers (Most Popular)

**Idea:** First place all non-zero values, then fill zeros.

```text
insertPos = 0

for i = 0 to n-1
    if nums[i] != 0
        nums[insertPos] = nums[i]
        insertPos++

while insertPos < n
    nums[insertPos] = 0
    insertPos++
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```

---

## 4. Single Pass Two Pointers with Swap (Optimal)

**Idea:** Maintain position of next non-zero element.

```text
left = 0

for right = 0 to n-1

    if nums[right] != 0

        swap nums[left], nums[right]

        left++
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```

---

## 5. Stable Partition

**Idea:** Partition array into non-zero and zero parts while preserving order.

```text
write = 0
for read = 0 to n-1
    if nums[read] != 0
        temp = nums[read]
        nums[read] = 0
        nums[write] = temp
        write++
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```

---

## 6. Queue Based

**Idea:** Store indices of zeros and swap when non-zero arrives.

```text
create queue
for i = 0 to n-1
    if nums[i] == 0
        push i to queue

    else if queue not empty
        zeroIndex = pop queue
        swap nums[zeroIndex], nums[i]
        push i to queue
```

### Complexity

```text
Time  : O(n)
Space : O(k)
```

where k = number of zeros.

---

## 7. Bubble Zeroes to End

**Idea:** Similar to Bubble Sort.

```text
repeat n times
    for i = 0 to n-2
        if nums[i] == 0
            swap nums[i], nums[i+1]
```

### Complexity

```text
Time  : O(n²)
Space : O(1)
```
