# Title
Remove Duplicates from Sorted Array

## Difficulty
Easy

## Description
Given a sorted integer array nums, remove the duplicates in-place such that each unique element appears only once and returns the new length. Do not allocate extra space for another array; modify the input array in-place with O(1) extra memory.

## Examples

### Example 1
```text
Input: [1,1,2]
Output: 2, [1,2]
Explanation: The function returns length = 2, and the first two elements of nums are 1 and 2.
```

### Example 2
```text
Input: [0,0,1,1,1,2,2,3,3,4]
Output: 5, [0,1,2,3,4]
Explanation: The function returns length = 5, and the first five elements are 0,1,2,3,4.
```

## Constraints
```text
0 <= nums.length <= 3 * 10^4
-10^4 <= nums[i] <= 10^4
nums is sorted in non-decreasing order.
Modify array in-place with O(1) extra memory.
```

---

## Idea
Use two pointers (slow-write and fast-read) to overwrite duplicates with the next unique element.

---

## Pseudo Code
```text
function removeDuplicates(nums)
    if nums.length == 0
        return 0
    write = 1
    for read from 1 to nums.length - 1
        if nums[read] != nums[read - 1]
            nums[write] = nums[read]
            write = write + 1
    return write
```

---

## Complexity

| Metric           | Value  |
| ---------------- | ------ |
| Time Complexity  | O(n)   |
| Space Complexity | O(1)   |

---

## Code Explanation

### Dry Run

| Step | read | write | nums state              | Operation                          | Result            |
| ---- | ---- | ----- | ------------------------| ---------------------------------- | ----------------- |
| 1    | 1    | 1     | [1,1,2]                 | nums[1] == nums[0]                 | skip              |
| 2    | 2    | 1     | [1,1,2]                 | nums[2] != nums[1]                 | nums[1]=2; write=2|
| End  | -    | 2     | [1,2,2]                 | return write                       | length = 2        |

1. Initialize write pointer at index 1.
2. Iterate read from 1 to n-1.
3. When current differs from previous, copy to write and increment write.
4. Returned write is the count of unique elements; first write elements contain uniques.

## Other Patterns

### Pattern 1

**Idea**
Use an extra array to collect unique elements then copy back.

**Pseudo Code**
```text
function removeDuplicatesExtra(nums)
    if nums.length == 0
        return 0
    result = empty list
    for i from 0 to nums.length - 1
        if result is empty or result[last] != nums[i]
            append nums[i] to result
    for i from 0 to result.length - 1
        nums[i] = result[i]
    return result.length
```

### Pattern 2

**Idea**
Use language/library utilities: unique in C++ or itertools.groupby in Python.

**Pseudo Code**
```text
function removeDuplicatesLibrary(nums)
    if nums.length == 0
        return 0
    new_end = library_unique(nums)   // returns end iterator or new length
    return new_end_length
```

## Comparison

| Approach            | Idea                                      | Time  | Space | Recommended |
| ------------------- | ----------------------------------------- | ----- | ----- | ----------- |
| Two Pointers        | Overwrite duplicates in-place             | O(n)  | O(1)  | Yes         |
| Extra Array         | Collect uniques then copy back            | O(n)  | O(n)  | No          |
| Library / Unique    | Use built-in unique/grouping utilities    | O(n)  | O(1)–O(n) | Yes (if allowed) |
