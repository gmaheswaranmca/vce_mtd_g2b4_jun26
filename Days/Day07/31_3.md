# Longest Substring Without Repeating Characters

## Difficulty
Medium

## Description
Given a string s, find the length of the longest substring without repeating characters.

## Examples

### Example 1
```text
Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with the length of 3.
```

### Example 2
```text
Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.
```

### Example 3
```text
Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3.
```

## Constraints
```text
0 <= s.length <= 10^5
s consists of English letters, digits, symbols and spaces.
Expected time: O(n) for optimal approach.
```

---

## Idea
Use a sliding window with a hash map that stores the last index of each character to move the left boundary directly past duplicates.

---

## Pseudo Code
```text
function lengthOfLongestSubstring(s)
    if s is empty return 0
    create map lastIndex
    left = 0
    maxLen = 0
    for right from 0 to length(s)-1
        ch = s[right]
        if ch in lastIndex and lastIndex[ch] >= left
            left = lastIndex[ch] + 1
        lastIndex[ch] = right
        maxLen = max(maxLen, right - left + 1)
    return maxLen
```

---

## Complexity

| Metric           | Value                |
| ---------------- | -------------------- |
| Time Complexity  | O(n)                 |
| Space Complexity | O(min(n, charset))   |

## Other Patterns

### Pattern 1

**Idea**
Sliding window with a HashSet and two pointers; expand right until duplicate, then shrink left removing chars until duplicate resolved.

**Pseudo Code**
```text
function lengthOfLongestSubstring_Set(s)
    create set window
    left = 0
    maxLen = 0
    for right from 0 to length(s)-1
        while s[right] in window
            remove s[left] from window
            left = left + 1
        add s[right] to window
        maxLen = max(maxLen, right - left + 1)
    return maxLen
```

### Pattern 2

**Idea**
Brute force: check all substrings and validate uniqueness using a set.

**Pseudo Code**
```text
function lengthOfLongestSubstring_Brute(s)
    n = length(s)
    maxLen = 0
    for i from 0 to n-1
        create set seen
        for j from i to n-1
            if s[j] in seen break
            add s[j] to seen
            maxLen = max(maxLen, j - i + 1)
    return maxLen
```

## Comparison

| Approach | Idea | Time | Space | Recommended |
| -------- | ---- | ---- | ----- | ----------- |
| HashMap sliding window | Track last index to jump left directly | O(n) | O(min(n, charset)) | Yes |
| HashSet two pointers | Remove until duplicate resolved | O(n) amortized | O(min(n, charset)) | Yes (simpler) |
| Brute force | Check all substrings | O(n^2) or O(n^3) depending on check | O(min(n, charset)) per check | No |
