# 3. Combinations

Idea: Choose k elements from n elements.

---

## Pseudocode

```text
 function bt(start)
    if len(path) == k
        res.add(clone(path))
        return
    for i <- start..n
        path.add(i)
        bt(i + 1)
        path.removeLast()            
function combine(n, k)
    res = [] // global
    path = [] //global
    bt(1)
    return res
```
