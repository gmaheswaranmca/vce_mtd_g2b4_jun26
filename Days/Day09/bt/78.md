# 1. Subsets
Idea: For every element -> * Include it * Exclude it
---
## Pseudocode
```text
subsets=[]
subset=[]
FUNCTION bt(index)
    IF index == n
        subsets.add(clone(subset))
        return
    subset.add(nums[index])
    bt(index+1)
    subset.removeLast()
    bt(index+1)
END FUNCTION

bt(0)
```

