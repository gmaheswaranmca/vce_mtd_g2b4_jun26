# 2. Permutations

Idea : Every position can choose any unused element.
---
## Pseudocode

```text
function bt(nums)
    if len(path) == len(nums)
        res.add(clone(path))
        return
    for i <- 1.. len(nums)-1
        if used[i]
            continue
        used[i] = true,  path.add(nums[i])
        bt(nums)
        path.removeLast(), used[i] = false   
function permute(nums)
    res = []                    //global
    used = [false for each element]  //global
    bt(nums)
    return res
```
