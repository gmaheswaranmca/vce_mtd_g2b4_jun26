# Add Two Numbers

## Difficulty
Medium

## Description
Add two non-empty singly linked lists representing non-negative integers in reverse digit order; return the sum as a linked list in reverse digit order. Each node contains a single digit. The lists do not contain leading zeros except the number 0 itself.

## Examples

### Example 1

```text
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807 -> [7,0,8]
```

### Example 2

```text
Input: l1 = [0], l2 = [0]
Output: [0]
Explanation: 0 + 0 = 0
```

### Example 3

```text
Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
Explanation: Handles carry propagation across different lengths.
```

## Constraints

```text
The number of nodes in each linked list is in the range [1, 100].
0 <= Node.val <= 9
Lists represent integers in reverse order.
Return list should represent the sum in reverse order.
Do not modify input lists.
```

---

## Idea
Digit-by-digit add corresponding nodes with a carry, create result nodes on the fly.

---

## Pseudo Code

```text
function addTwoNumbers(l1, l2)
    create dummy = new ListNode(0)
    set tail = dummy
    set carry = 0
    while l1 != null OR l2 != null OR carry != 0
        set v1 = (l1 != null) ? l1.val : 0
        set v2 = (l2 != null) ? l2.val : 0
        set sum = v1 + v2 + carry
        set carry = sum / 10
        set digit = sum % 10
        set tail.next = new ListNode(digit)
        set tail = tail.next
        if l1 != null then set l1 = l1.next
        if l2 != null then set l2 = l2.next
    return dummy.next
```

## Other Patterns

### Pattern 1

**Idea**

Recursive digit-by-digit addition carrying value to next call.

**Pseudo Code**

```text
function addTwoNumbersRec(l1, l2)
    return addRec(l1, l2, 0)

function addRec(l1, l2, carry)
    if l1 == null AND l2 == null AND carry == 0 then return null
    set v1 = (l1 != null) ? l1.val : 0
    set v2 = (l2 != null) ? l2.val : 0
    set sum = v1 + v2 + carry
    set carryNext = sum / 10
    set node = new ListNode(sum % 10)
    set node.next = addRec(l1 != null ? l1.next : null, l2 != null ? l2.next : null, carryNext)
    return node
```

### Pattern 2

**Idea**

Convert lists to integers, add, and rebuild list (only safe when numbers fit in available integer type).

**Pseudo Code**

```text
function addByConversion(l1, l2)
    set num1 = 0
    set place = 1
    while l1 != null
        set num1 = num1 + l1.val * place
        set place = place * 10
        set l1 = l1.next
    set num2 = 0
    set place = 1
    while l2 != null
        set num2 = num2 + l2.val * place
        set place = place * 10
        set l2 = l2.next
    set total = num1 + num2
    if total == 0 then return new ListNode(0)
    create dummy = new ListNode(0)
    set tail = dummy
    while total > 0
        set tail.next = new ListNode(total % 10)
        set tail = tail.next
        set total = total / 10
    return dummy.next
```

## Comparison

| Approach        | Idea                                  | Time         | Space            | Recommended |
| --------------- | ------------------------------------- | ------------ | ---------------- | ----------- |
| Iterative (main)| Digit-by-digit with carry             | O(max(n,m))  | O(max(n,m))      | Yes         |
| Recursive       | Same as iterative using recursion     | O(max(n,m))  | O(max(n,m)) stack| Yes (small lists) |
| Conversion      | Convert to integers, add, rebuild     | O(n+m)       | O(n+m)           | No (overflow risk) |
