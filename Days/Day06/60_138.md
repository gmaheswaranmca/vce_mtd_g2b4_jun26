# Copy List with Random Pointer (LeetCode 138)

## Difficulty
Medium

## Description
Deep copy a singly linked list where each node has two pointers: next and random. Return the head of the cloned list such that both next and random pointers are correctly assigned in the clone.

## Examples

### Example 1

```text
Input: head = [[7,null],[13,0],[11,4],[10,2],[1,0]]
Output: [[7,null],[13,0],[11,4],[10,2],[1,0]]
Explanation: Each element is [val, random_index] where random_index is the index of the node (0-based) pointed by random or null.
```

### Example 2

```text
Input: head = [[1,1],[2,1]]
Output: [[1,1],[2,1]]
Explanation: First node's random -> second node; second node's random -> second node.
```

## Constraints

```text
0 <= n <= 1000
-10000 <= Node.val <= 10000
random pointer may be null or point to any node in the list
You must return a deep copy; no node from the original list should be referenced.
```

---

### Pattern 1

**Idea**

Use a HashMap to map original nodes to their clones and assign next and random using the map.

**Pseudo Code**

```text
function copyRandomList(head)
    if head == null
        return null
    map = empty hash map
    curr = head
    while curr != null
        map[curr] = new Node(curr.val)
        curr = curr.next
    curr = head
    while curr != null
        map[curr].next = map.get(curr.next, null)
        map[curr].random = map.get(curr.random, null)
        curr = curr.next
    return map[head]
```

## Other Patterns
## Idea
Interleave cloned nodes with original nodes to set random pointers in O(n) time and O(1) extra space, then detach clones.

---

## Pseudo Code

```text
function copyRandomList(head)
    if head == null
        return null
    // Step 1: Interleave cloned nodes
    curr = head
    while curr != null
        clone = new Node(curr.val)
        clone.next = curr.next
        curr.next = clone
        curr = clone.next
    // Step 2: Assign random pointers for clones
    curr = head
    while curr != null
        if curr.random != null
            curr.next.random = curr.random.next
        else
            curr.next.random = null
        curr = curr.next.next
    // Step 3: Detach clones to form the new list
    curr = head
    cloneHead = head.next
    while curr != null
        clone = curr.next
        curr.next = clone.next
        if clone.next != null
            clone.next = clone.next.next
        curr = curr.next
    return cloneHead
```

## Complexity

| Metric           | Value   |
| ---------------- | ------- |
| Time Complexity  | O(n)    |
| Space Complexity | O(1) extra (O(n) output) |

## Code Explanation

### Dry Run

| Step | State | Operation | Result |
| ---- | ----- | --------- | ------ |
| 1 | Original: 1->2->3, randoms: 1.random->3,2.random->1,3.random->3 | Interleave clones after each original node | 1->1'->2->2'->3->3' |
| 2 | Interleaved list | Set clone.random = original.random.next for each original | 1'.random->3', 2'.random->1', 3'.random->3' |
| 3 | Clones randoms set | Separate lists by fixing next pointers | Original: 1->2->3, Clone: 1'->2'->3' (with correct randoms) |

## Comparison

| Approach     | Idea                                    | Time  | Space         | Recommended |
| ------------ | --------------------------------------- | ----- | ------------- | ----------- |
| Interleaving | Interleave clones, set random via next, detach | O(n)  | O(1) extra    | Yes         |
| HashMap      | Map old->new then assign next & random  | O(n)  | O(n) extra    | No (simple but more memory) |
