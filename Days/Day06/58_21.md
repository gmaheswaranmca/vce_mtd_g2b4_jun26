# Merge Two Sorted Lists

## Difficulty
Easy

## Description
Merge two sorted singly linked lists and return it as a single sorted list by splicing together the nodes of the first two lists.

## Examples

### Example 1

```text
Input: l1 = [1,2,4], l2 = [1,3,4]
Output: [1,1,2,3,4,4]
Explanation: Compare node values and append the smaller one to the merged list.
```

### Example 2

```text
Input: l1 = [], l2 = [0]
Output: [0]
Explanation: One list empty -> return the other list.
```

## Constraints

```text
0 <= total nodes (n) <= 10^5
-10^4 <= Node.val <= 10^4
Lists are sorted in non-decreasing order
```

---

## Idea
Use a dummy head and an iterative two-pointer merge selecting the smaller current node at each step.

---

## Pseudo Code

```text
function solve(l1, l2)
    if l1 is null then return l2
    if l2 is null then return l1
    create dummy node with value 0
    set current = dummy
    while l1 is not null and l2 is not null
        if l1.val <= l2.val
            set current.next = l1
            set l1 = l1.next
        else
            set current.next = l2
            set l2 = l2.next
        set current = current.next
    if l1 is not null then set current.next = l1
    if l2 is not null then set current.next = l2
    return dummy.next
```

## Complexity

| Metric           | Value     |
| ---------------- | --------- |
| Time Complexity  | O(n)      |
| Space Complexity | O(1)      |

## Code Explanation

### Dry Run

| Step | l1 (remaining) | l2 (remaining) | current points to | Operation | Merged list |
| ---- | -------------- | -------------- | ----------------- | --------- | ----------- |
| 1    | [1,2,4]        | [1,3,4]        | dummy             | compare 1 <= 1 -> take l1 | [1] |
| 2    | [2,4]          | [1,3,4]        | 1                 | compare 2 > 1 -> take l2 | [1,1] |
| 3    | [2,4]          | [3,4]          | last 1            | compare 2 <= 3 -> take l1 | [1,1,2] |
| 4    | [4]            | [3,4]          | last 2            | compare 4 > 3 -> take l2 | [1,1,2,3] |
| 5    | [4]            | [4]            | last 3            | compare 4 <= 4 -> take l1 | [1,1,2,3,4] |
| 6    | []             | [4]            | last 4            | l1 exhausted -> append l2 | [1,1,2,3,4,4] |

## Other Patterns

### Pattern 1

**Idea**

Recursive merging by selecting smaller head and recursing on the remainder.

**Pseudo Code**

```text
function solve(l1, l2)
    if l1 is null then return l2
    if l2 is null then return l1
    if l1.val <= l2.val
        set l1.next = solve(l1.next, l2)
        return l1
    else
        set l2.next = solve(l1, l2.next)
        return l2
```

## Comparison

| Approach   | Idea                                 | Time   | Space       | Recommended |
| ---------- | ------------------------------------ | ------ | ----------- | ----------- |
| Iterative  | Dummy head + two pointers            | O(n)   | O(1) (aux)  | Yes         |
| Recursive  | Pick smaller head and recurse        | O(n)   | O(n) (call stack) | No          |
