## 032. Reverse Linked List

| Item        | Value                                                               |
| ----------- | ------------------------------------------------------------------- |
| LC #        | 206                                                                 |
| Difficulty  | Easy                                                                |
| Title       | Reverse Linked List                                                 |
| Description | Reverse a singly linked list and return the new head.               |
| Pattern     | Linked List Pointer Reversal                                        |
| Idea        | Iteratively reverse next pointers using prev,current,next pointers. |

---

## Description

Given the `head` of a singly linked list, reverse the list and return the reversed list.

You must reverse the links between nodes so that:

```text
1->2->3->4->5

becomes

5->4->3->2->1
```

---

## Examples

```text
Input: head=[1,2,3,4,5]
Output: [5,4,3,2,1]

Explanation:
Reverse each next pointer one by one.
```

```text
Input: head=[1,2]
Output: [2,1]

Explanation:
2 points to 1 and 1 points to null.
```

```text
Input: head=[]
Output: []

Explanation:
Empty list remains empty.
```

---

## Constraints

```text
The number of nodes is in the range [0,5000].

-5000<=Node.val<=5000
```

---

## Idea (One Line)

Reverse the direction of each next pointer while traversing the linked list once.

---

## Pseudocode

```text
function reverseList(head)
    prev=null
    current=head
    while current!=null
        nextNode=current.next
        current.next=prev
        prev=current
        current=nextNode
    return prev
```

---

## Dry Run

### Input

```text
1->2->3->4->5->null
```

Initially

```text
prev=null
current=1
```

| Step | prev | current | nextNode | Action            |
| ---- | ---- | ------- | -------- | ----------------- |
| 1    | null | 1       | 2        | 1.next=null       |
|      | 1    | 2       | -        | Move prev,current |
| 2    | 1    | 2       | 3        | 2.next=1          |
|      | 2    | 3       | -        | Move prev,current |
| 3    | 2    | 3       | 4        | 3.next=2          |
|      | 3    | 4       | -        | Move prev,current |
| 4    | 3    | 4       | 5        | 4.next=3          |
|      | 4    | 5       | -        | Move prev,current |
| 5    | 4    | 5       | null     | 5.next=4          |
|      | 5    | null    | -        | Loop ends         |

Final:

```text
5->4->3->2->1->null
```

Return:

```text
prev=5
```

---

## Code Explanation

```text
1. Initialize prev as null.
2. Initialize current as head.
3. Store current.next in nextNode.
4. Reverse current.next to prev.
5. Move prev to current.
6. Move current to nextNode.
7. Continue until current becomes null.
8. prev becomes the new head.
9. Return prev.
```

---

## Complexity

```text
Time Complexity: O(n)

Space Complexity: O(1)
```

# Other Patterns

## Pattern 1: Recursive Reversal

### Idea

Reverse the remaining list recursively and attach current node at the end.

### Pseudocode

```text
function reverseList(head)

    if head==null or head.next==null
        return head

    newHead=reverseList(head.next)

    head.next.next=head

    head.next=null

    return newHead
```

---

## Pattern 2: Head Insertion Method

### Idea

Detach nodes one by one and insert them at the beginning of the reversed list.

### Pseudocode

```text
function reverseList(head)

    reversedHead=null

    while head!=null

        nextNode=head.next

        head.next=reversedHead

        reversedHead=head

        head=nextNode

    return reversedHead
```

---

## Pattern 3: Stack Based

### Idea

Push nodes into stack and rebuild list in reverse order.

### Pseudocode

```text
function reverseList(head)

    stack=emptyStack()

    current=head

    while current!=null

        push current into stack

        current=current.next

    newHead=pop stack

    current=newHead

    while stack not empty

        current.next=pop stack

        current=current.next

    current.next=null

    return newHead
```

---

## Pattern 4: Array Based

### Idea

Store nodes in array and reconnect from end to beginning.

### Pseudocode

```text
function reverseList(head)

    nodes=emptyList()

    current=head

    while current!=null

        add current to nodes

        current=current.next

    for i=length(nodes)-1 downto 1

        nodes[i].next=nodes[i-1]

    nodes[0].next=null

    return nodes[length(nodes)-1]
```

## Pattern Comparison

| Pattern                  | Idea                                   | Time |                Space | Recommended |
| ------------------------ | -------------------------------------- | ---: | -------------------: | ----------- |
| Iterative Three Pointers | Reverse next pointers while traversing | O(n) |                 O(1) | ⭐⭐⭐⭐⭐ Best  |
| Recursive                | Reverse remaining list recursively     | O(n) | O(n) recursion stack | ⭐⭐⭐⭐        |
| Head Insertion           | Insert nodes at front                  | O(n) |                 O(1) | ⭐⭐⭐⭐        |
| Stack Based              | Push then pop nodes                    | O(n) |                 O(n) | ⭐⭐          |
| Array Based              | Store nodes then reconnect             | O(n) |                 O(n) | ⭐⭐          |

---

## Important Observation

```text
Only three pointers are required:

prev
current
nextNode

Example:

Before:
prev=null
current=1->2->3

After first iteration:
prev=1->null
current=2->3

After second iteration:
prev=2->1->null
current=3

Finally:
prev=5->4->3->2->1->null
```

**The Iterative Three Pointer approach is the standard and most frequently asked solution in interviews and LeetCode.**
