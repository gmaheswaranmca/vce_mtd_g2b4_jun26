## 030. Delete the Middle Node of a Linked List
| Field       | Value                                                                                       |
| ----------- | ------------------------------------------------------------------------------------------- |
| LC #        | 2095                                                                                        |
| Title       | Delete the Middle Node of a Linked List                                                     |
| Difficulty  | Medium                                                                                      |
| Category    | Linked List, Fast & Slow Pointer                                                            |
| Description | Delete the middle node of a linked list and return the modified list.                       |
| Idea        | Use slow and fast pointers to locate the middle node and its previous node, then remove it. |

---

## Description
```text
You are given the `head` of a singly linked list.
Delete the **middle node** and return the head of the modified linked list.
The middle node of a linked list of size `n` is the `⌊n/2⌋th` node using **0-based indexing**.
For:
n=1 -> delete node at index 0
n=2 -> delete node at index 1
n=3 -> delete node at index 1
n=4 -> delete node at index 2
```

---

## Example 1

```text
Input:head=[1,3,4,7,1,2,6]
Output:[1,3,4,1,2,6]
Explanation:
Length=7
Middle index=7/2=3
Node to delete=7
Result:1->3->4->1->2->6
```

---

## Example 2

```text
Input:head=[1,2,3,4]
Output:[1,2,4]
Explanation:
Length=4
Middle index=4/2=2
Delete node 3.
```

---

## Example 3

```text
Input:head=[2,1]
Output:[2]
Explanation:
Middle index=2/2=1
Delete node 1.
```

---

## Constraints

```text
1<=number of nodes<=10^5
1<=Node.val<=10^5
```

---

## Idea (One Line)

```text
Move slow by 1 step and fast by 2 steps; when fast reaches end, slow points to middle.
```

---

## Optimal Pattern: Fast and Slow Pointer
### Pseudo Code
```text
function deleteMiddle(head)
    if head==null
        return null
    if head.next==null
        return null
    slow=head
    fast=head
    prev=null
    while fast!=null and fast.next!=null
        prev=slow
        slow=slow.next
        fast=fast.next.next
    prev.next=slow.next
    return head
```

## Code Explanation

```text
1. If list contains only one node, return null.
2. Initialize slow=head,fast=head and prev=null.
3. Move slow one step.
4. Move fast two steps.
5. Keep prev one node behind slow.
6. When fast reaches end, slow points to middle node.
7. Remove middle by prev.next=slow.next.
8. Return head.
```

# Other Patterns
## Pattern 1: Count Nodes Then Delete
### Idea
```text
Count total nodes, compute middle index and delete it in second traversal.
```

### Pseudo Code
```text
function deleteMiddle(head)
    if head.next==null
        return null
    n=0
    temp=head
    while temp!=null
        n=n+1
        temp=temp.next
    middle=n/2
    temp=head
    for i=0 to middle-1
        prev=temp
        temp=temp.next
    prev.next=temp.next
    return head
```

## Pattern 2: Fast and Slow Pointer
### Idea
```text
Use slow and fast pointers to locate middle in a single traversal.
```

### Pseudo Code
```text
function deleteMiddle(head)
    if head.next==null
        return null
    slow=head
    fast=head
    prev=null
    while fast!=null and fast.next!=null
        prev=slow
        slow=slow.next
        fast=fast.next.next
    prev.next=slow.next
    return head
```

---

## Pattern 3: Dummy Node
### Idea
```text
Use dummy node before head to simplify deletion logic.
```

### Pseudo Code
```text
function deleteMiddle(head)
    dummy=new Node(0)
    dummy.next=head
    slow=dummy
    fast=dummy
    prev=dummy
    while fast.next!=null and fast.next.next!=null
        prev=slow
        slow=slow.next
        fast=fast.next.next
    slow.next=slow.next.next
    return dummy.next
```
